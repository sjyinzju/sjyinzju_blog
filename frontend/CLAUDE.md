# 项目全局技术栈与编码规范

## 技术栈 (Tech Stack)
- 框架：Next.js 14 (严格使用 App Router 架构)
- 语言：TypeScript (要求严格的类型定义，禁用 `any`)
- 样式：Tailwind CSS
- UI 库：Shadcn UI (优先复用基础组件，避免重复造轮子)
- 动效：Framer Motion (仅用于提升质感的微交互，克制使用)

## 编码与输出准则 (Core Principles)
1. **务实第一**：在执行编程、报错修改和输出代码时，必须严谨、完整地审视一遍代码。只给出扎实、不出错的实现，不强求所谓的高级模式或过度封装。
2. **结构优先**：界面构建必须采用清晰的语义化 HTML 标签和标准的 Flex/Grid 布局。
3. **渐进式修改**：每次只输出当前需要修改的代码块或文件，不要毫无意义地重写整个文件。
